# HackFS 2023

[HackFS 2023](https://ethglobal.com/events/hackfs2023) is an event bringing together some of the top minds and experts in Ethereum Ecosystem. We enable teams to make something great in only 3 weeks by providing an abundance of web3 resources like mentors, sponsors, and software. Additionally, Push Protocol had sponsored **$5,000 in bounties** for any projects that successfully integrates the Push SDK.

## Winners

### _Chitty chat_

Chitty Chat is an advanced Push App with web3 integration, offering a ChatGPT-inspired command interface for easy FVM DataDao management, IPFS file handling, and Bacalhau Stable Diffusion jobs. It requires Apecoin subscription for ChatGPT services, supports ENS name resolution, and deploys its server-side via Spheron for scalability.

MintLock provides a secure platform for encrypting NFTs, protecting the metadata and digital assets of creators and collectors. It emphasizes privacy, with end-to-end encryption and decentralized storage, ensuring that only authorized users can access sensitive information, thereby redefining digital ownership security.

Flight is a decentralized hub for the FPV drone community, offering a space for pilots and enthusiasts to connect, chat, plan races, share content, and embark on virtual expeditions.
