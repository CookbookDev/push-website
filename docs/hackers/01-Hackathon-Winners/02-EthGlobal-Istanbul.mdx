# EthGlobal Istanbul 2023

[ETHGlobal Istanbul](https://ethglobal.com/events/istanbul), from November 17-19, invited 1500 global Ethereum builders for a hackathon at ICC. With a $500,000 prize pool, it was set against the city's unique Asia-Europe crossroads backdrop, following Devconnect 2023. Push Protocol had sponsored **$5,000 in bounties** for any projects that successfully integrates the Push SDK.
